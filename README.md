The frontend of the Blog Application is built using React.js and Vite to provide a fast, responsive, and interactive user experience. It allows users to browse blogs, authenticate securely, create and manage articles, and interact with the platform through a clean and modern interface.
The frontend of the Blog Application is developed using React.js and Vite to deliver a fast, responsive, and user-friendly experience. It provides interactive interfaces for users, authors, and admins to access the platform efficiently. The frontend includes features such as authentication, protected routes, article creation and management, dynamic navigation, and seamless API integration with the backend. It is designed with responsive layouts to ensure smooth performance across desktop and mobile devices while maintaining a clean and modern user interface.

Features
User Login and Registration
Role-based Dashboard
Create, Edit, and Delete Blogs
Responsive Design
Protected Routes
API Integration with Backend
Dynamic Routing
Authentication using JWT
State Management
Fast Performance with Vite
Technologies Used
React.js
Vite
Axios
React Router DOM
Bootstrap / CSS
Installation
cd blog-frontend
npm install
npm run dev
Environment Variables

Create a .env file:

VITE_API_URL=https://capstone-frontend-theta-nine.vercel.app/
Deployment

Frontend deployed using Vercel.
